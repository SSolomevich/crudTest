Cтандартное crud приложение, которое отображает список всех юзеров в базе. 
Имеется возможность удаления, редактирования, добавления новых и поиска по уже существующим юзеров.
Поиск записей осуществляется по id.
Реализован пейджинг. 
Используемые технологии: Maven, Hibernate, Spring, Tomcat, MySQL, Spring MVC. 
Скрипт БД в test.sql
